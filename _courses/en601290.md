This course will provide students with a rich development experience, focused on the design and implementation of user interfaces and mobile applications. A brief overview of human computer interaction will provide context for designing, prototyping and evaluating user interfaces. Students will invent their own mobile applications and implement them using the Android SDK. An overview of the Android platform and available technologies will be provided, as well as XML for layouts, and general concepts for effective mobile development.
